# Alarmierte Einsatzmittel

Dieses Widget zeigt die alarmierten Einsatzmittel an.


![](/img/image-43.png)



```
    {
        "cols": 1,
        "rows": 3,
        "y": 1,
        "x": 2,
        "type": "operation-gear"
    }
```
