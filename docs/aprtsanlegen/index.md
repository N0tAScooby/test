# APRTs anlegen

Durch das Einpflegen der ARPTs können diese als alarmierbare Einsatzmittel in den Alarmierungsprofilen und auf dem Wallboard genutzt werden.



## Gerät anlegen



Den Menüpunkt **Inventar** auswählen, dann links auf **Inventar** und **neu anlegen**.



Fülle hier nun die folgenden Felder aus:



* Organisation auswählen
* Inventargruppe auswählen (diese kann z.B. "APRT Digitalfunkmelder" heißen - **Wichtig:** Die Gruppe sollte schon existieren, falls nicht frag bei deinem Administrator nach)
* Status auf **aktiv** schalten
* Bezeichnung vergeben (z.B. Max Mustermann APRT)
* Beschaffungsdatum angeben


![](/img/Bildschirmfoto-2022-04-21-um-16.21.57-1024x810.png)



Klicke nun unten auf **Alarmierung** in den Inventargruppen-Details.



* Die Checkbox **NICHT** aktivieren
* Den Bezeichner freilassen
* ISSI angeben (im Normalfall 7-Stellen)
* Endgeräte-Typ: **APRT (Digitalfunkmelder)** auswählen


![](/img/Bildschirmfoto-2022-04-21-um-16.27.23-1024x390.png)



Klicke jetzt auf Speichern, um das Gerät hinzuzufügen.



## Inventargruppe anlegen - nur Admins



Falls bis jetzt noch keine Inventargruppe für die ARPTs angelegt wurde, sollte dies jetzt nachgeholt werden. Dafür den Menüpunkt Inventar wählen, dann links Inventargruppen und neu anlegen.



Hier nun einen beliebigen **Namen**, **Organisation** und **verantwortliche Rolle** auswählen und die Option **alarmierbare Einsatzmittel** aktivieren. Wird die Checkbox nicht aktiviert, fehlt der Reiter **Alarmierung** unten in den Details.
